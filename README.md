# AI Study Buddy
AI Study Buddy is an AI-powered educational web application that helps users understand topics through structured explanations generated using Google Gemini.

## Features
- AI-generated topic explanations
- Clean and responsive interface
- Google Gemini API integration
- Copy response functionality

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Gemini API

## Setup
1. Get a Gemini API key from Google AI Studio.
2. Replace `YOUR_API_KEY` in `index.html` with your own API key.
3. Open `index.html` in a browser.

## Security Note
The API key is not included in this repository for security reasons.
